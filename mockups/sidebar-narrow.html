Sidebar Narrow - Design Notes

Width: 64px narrow vs 256px normal. The narrow sidebar uses icon-only navigation with tooltips on hover.

Tooltips: Hover over any icon in the narrow sidebar to see the module name. Uses a directional tooltip with arrow pointing to the icon.

Group popover: The "Production" group shows a popover on hover with sub-items (Industrie, Shopping List, Planetary). This replaces the collapsible accordion from the normal sidebar.

Active state: Active items use the same cyan gradient background + pulse dot. The group icon has a small chevron indicator at bottom-right showing it has sub-items.

Scan line effect: Same hover animation as the normal sidebar (gradient sweep from left to right).

User area: Compact avatar with stacked icon buttons (Characters, Settings, Logout) + version number. Hover reveals a popover with full user info and labeled actions.

Separator: A subtle gradient line separates main navigation from the Admin item, replacing the vertical spacing from the normal sidebar.

Toggle button: A small circular button on the right edge of each sidebar indicates collapsibility. In implementation, clicking would animate between the two states.

Try It

1. Hover any icon in the narrow sidebar to see the tooltip.

2. Hover the Production icon (beaker/flask, 3rd from top) to see the sub-item popover with Industrie, Shopping List, and Planetary.

3. Hover the avatar area at the bottom of the narrow sidebar to see the user popover with name, corp, and action buttons.

4. Compare both sidebars side by side -- same colors, same icons, same active states.
